# Concept — Study Diary

## One-liner
A study diary that improves your studying. You journal repeatedly through a messenger-style interface; an agent analyzes what you put in, watches your study patterns, and uses what it learns to generate quizzes from your own study materials.

## Two interacting features

### 1. Journaling (messenger-style)
- User journals repeatedly through a chat/messenger interface.
- Each entry becomes part of a growing personal diary.
- Low-friction, short-form, conversational — not a blank-page writing exercise.

### 2. Agent analysis & quizzing
- The agent reads the accumulated diary context.
- Watches study patterns across entries (what the user studied, when, how, how it went).
- Helps the user identify the study patterns that are most effective for them.
- After the user feeds in study materials, the agent generates quizzes based on those materials.

## How the two interact
- The journaling feeds the agent the context about *how* the user studies.
- The study materials feed the agent the content the user is studying.
- Quizzes are shaped by both: what the materials contain, and what the diary reveals about how the user learns best.
